Shell Commands

Sessions (prefix +)

ddetach $rename session slist/switch sessions Ctrl+ssave (resurrect) Ctrl+rrestore (resurrect)
Auto-saves every 10 min via continuum

Windows (prefix +)

cnew window n / pnext / prev 0-9go to window # ,rename &close window < / >swap left / right
No prefix: Ctrl+h / Ctrl+l for prev/next

Panes (prefix +)

|split vertical -split horizontal h j k lnavigate (vim-style) H J K Lresize zzoom (fullscreen) xclose pane Spacecycle layouts { / }swap pane position

Copy Mode (prefix +)

[enter copy mode vstart selection ycopy selection ]paste Escapeexit copy mode
Uses vim keys: h j k l to move, / to search
