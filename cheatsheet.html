VIM CHEATSHEET

WebStorm to Neovim - Level Up Guide

üß† The Mindset

Vim is a language. You speak commands to your editor.

verb + motion = action
d (delete) + w (word) ‚Üí delete word
c (change) + i" (inside quotes) ‚Üí change inside quotes
y (yank) + $ (end of line) ‚Üí copy to end of line

‚ö° Verbs (Actions)

d Delete (and copy to register)
c Change (delete and enter insert mode)
y Yank (copy)
v Visual select
> Indent right
< Indent left
gu Lowercase
gU Uppercase

üì¶ Text Objects

Use with i (inside) or a (around)

" ' ` Quotes

{ } B Braces

üöÄ Motions (Movement)

w / b Next / previous word
e End of word
0 / $ Start / end of line
^ First non-blank char
f{char} Jump TO character
t{char} Jump BEFORE character
% Matching bracket
{ / } Prev / next paragraph
gg / G Top / bottom of file
Ctrl+d / Ctrl+u Half page down / up

üî¢ Relative Line Numbers

With relative numbers, just look at the line and use that number!

Command Action
5j Jump 5 lines down
12k Jump 12 lines up
d8j Delete cursor + 8 lines down
d5k Delete cursor + 5 lines up
y12j Yank cursor + 12 lines below
c7j Change cursor + 7 lines below
>5j Indent cursor + 5 lines below
<3k Unindent cursor + 3 lines above
V8j Visual line select 8 lines down
gq10j Format cursor + 10 lines below
=8j Auto-indent cursor + 8 lines
The pattern: [operator][count]j/k ‚Äî Look at the relative number, type that number with j (down) or k (up), add an operator in front to act on those lines

TMUX Terminal Multiplexer

üöÄ Getting Started

Prefix key is Ctrl+a (press, release, then command)

tmux Start new session
tmux new -s work Start named session
tmux ls List sessions
tmux a Attach to last session
tmux a -t work Attach to named session
tmux kill-session -t x Kill session

üìÅ Sessions (prefix +)

d Detach from session
$ Rename session
s List/switch sessions
Ctrl+s Save session (resurrect)
Ctrl+r Restore session (resurrect)
Auto-save: Sessions save every 10 minutes via continuum plugin

ü™ü Windows (prefix +)

c Create new window
n / p Next / previous window
0-9 Go to window #

& Close window

No prefix: Ctrl+h / Ctrl+l for prev/next window

üìê Panes (prefix +)

üìã Copy Mode (prefix +)

Copy mode uses vim keybindings: h/j/k/l to move, / to search
